set_allow_myself

set_allow_myself(a_bool)

If my self(a script or an application running FinderSelection) is selected in Finder, whether my selft is treated as a member of selection or not.

Parameters
  • a_bool (boolean) : If true is given, the result of get_selection may include the application. The default is false.
Result

script object : me
